Vagrant Setup Tips

for Windows Users

Unfortunately, Vagrant installation to Windows is quite different from Linux and Mac OS. Here, we describe installation tips for you to successfully install Vagrant.

Using id/password with an arbitrary SSH client

You can enter the following settings to your favorite SSH client whenever you need to ssh to a Vagrant instance:

  1. user id: vagrant
  2. password: vagrant
  3. port: 2222

id file based login with PuTTY

Alternatively, you can skip entering user id/password by specifying Vagrant's id file. We describe how to use the id file with PuTTY.

Importing Vagrant Private Key with PuTTYGen

Download PuTTYGen from this site and save your local disk. Then run the downloaded puttygen.exe executable file.
After opening PuTTYGen window, choose Conversions > Import Key from the top menu.
Then you need to choose Vagrant's provided private key at %HOMEDRIVE%\%HOMEPATH\.vagrant.d\insecure_private_key. For many cases, by default, %HOMEDRIVE% is C: and %HOMEPATH% is \Documents and Settings\your-user-name.
Finally, select SSH2-RSA as Parameters, and 1024 as Number of bits....

Generating Keypair

Next, you need to generate a keypair with the imported private key. Select Key > Generate Key Pair from the menu bar.
Move mouse cursor to create random number and enter comment and passphrase as shown below. 


Here you need to enter the following items:

  1. Key comment ... any text
  2. Key passphrase ... used for generating a private key, any text

Note that the passphrase is used for sealing the generated private key. Do not forget it, or you need to re-generate the private key.

Starting PuTTY

Now the time to run PuTTY, download PuTTY from the same site and save your local disk.
Then, run putty.exe and you will see the following window:

Select Connection > SSH > Auth from the Category pane on the left and you can specify the generated PuTTY key file on the right pane.
