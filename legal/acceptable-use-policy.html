IIDN Services Quota (BETA)

Your applications cannot exceed the following quota.

Models

Up to 5 Model types are allowed to be defined per account.
The maximum number of fields per model type is 20 (uid and rev fields are not included).
The total length of field values must be within 300.

Applications

You can create 1 application with up to 5 packages.
The size of each server application package (MOAT js and package.json) must be within 30,000
bytes in zip archive format.
You can upload up to 10 client application packages (MOAT Java/C and/or package.json).
