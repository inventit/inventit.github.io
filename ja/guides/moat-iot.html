MOAT IoT?

MOAT IoT は、ServiceSync プラットフォーム上で動作する IoT/M2M アプリケーションを構築するための APIセット及び、その仕様です。

このセクションでは、IoT/M2M アプリケーションを開発のための、一貫した結合方法を説明します。

ServiceSync が提供する API セットで、以下のことができるようになります。

  • Web 上で IoT サービスを構築することができます。
  • あなたの既存のサービスに、新しい機能を追加することができます。
  • あなたが本来構築したいサービス開発に集中することができます。

まずは、概要を確認しましょう。
