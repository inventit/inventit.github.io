プラットフォーム

開発者が自由に利用できる Sandbox プラットフォームを提供します。
ServiceSync は、デバイスとプラットフォーム間のセキュアな通信、柔軟なデータモデリング手段、そしてサーバーサイドで動作する Javascript の実行環境を提供します。

詳細  

MOAT-IoT

MOAT IoT は、ServiceSyncが提供する IoT/M2M プラットフォームにアクセスするための API セットです。
この API セットは、プラットフォームと対話するための REST API、デバイスを制御するサーバサイド Javascript API、 そして組み込み開発を容易にする Java/C API から構成されます。
